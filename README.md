# AI-shape-Detector
AI-based Shape Detection using OpenCV &amp; Machine Learning
🎯 AI Shape Detector

An intelligent computer vision project that detects and classifies geometric shapes in real-time using machine learning and image processing techniques.

🚀 Features

🔍 Real-time shape detection using webcam
🧠 Machine learning-based classification
✋ Hand tracking using MediaPipe
📊 Accuracy evaluation with confusion matrix
⚡ Fast and efficient detection
🛠️ Tech Stack
Python
OpenCV
NumPy
Scikit-learn
MediaPipe

⚙️ Installation

Create virtual environment:
python -m venv venv
venv\Scripts\activate   # Windows
Install dependencies:
pip install -r requirements.txt

▶️ Usage

Run the main file:

python App.py

🧠 How It Works
Capture image using webcam
Preprocess image (grayscale, thresholding)
Detect contours
Extract features
Classify shapes using ML model

📊 Model Performance
Accuracy: 96%+ (depends on dataset)
Evaluated using confusion matrix
